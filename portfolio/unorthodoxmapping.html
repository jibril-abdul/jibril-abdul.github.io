Unorthodox Mapping

THE PROBLEM

There is a need for people to find the ideal neighborhood in which they would like to live.

THE SOLUTION

I designed an application that would allow users to select certain 'unorthodox' criteria. These selections were then used to generate possible neighborhoods the user might enjoy living in.

Adobe XD
Design & Frontend (HTML5/CSS3)
Backend Development (NodeJS)

I served as the wildcard in a 6 person team. Not only did I design the application, but I also assisted on both the front end (HTML5, CSS3, & Angular 6) as well as the back-end (NodeJS) teams. We worked with an Agile Scrum development process - delivering an MVP every week.

STRATEGY

Our team first had to understand our app's goals, define an ideal user, and then undertand that user's goals.

RESEARCH

Reviewed potential competitor web applications. These applications helped influence the design as I was able to implement some established design patterns

DESIGN

The application was designed using AdobeXD. This allowed the team to quickly prototype.
